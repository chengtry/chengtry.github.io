项目背景

IBMJDK1.6、Spring、MyBatis、Tomcat7(tomcat直接启动项目，不是maven项目)

启动报错

Error:java: Illegal character in scheme at index 0 ：e:///D:/work

解决

一开始看到这个报错很奇怪，这个是build project报的错，也就是说编译时就报错。报错显示找e盘下的D盘的东西，但是这台电脑没有e盘啊！！！于是代码全局搜索e:相关的东西，但是没有找到。

最后是找到这位老哥的文章才解决的：https://blog.csdn.net/Ccyy325108/article/details/122069938

这个是IDEA设置的问题，Use compiler from module target JDK when possible：尽可能使用来自模块目标JDK的编译器

项目中用到的是IBMJDK1.6，勾选这个是选择用于运行IntelliJ IDEA相同的JDK
